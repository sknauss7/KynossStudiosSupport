Frequently Asked Questions

How does Cublii work?
Each day, Cublii presents four spatial challenges. You see a flat 2D cube net with symbols on each face. Your job is to mentally fold it into a 3D cube and answer a question about the result — like which symbol is opposite another, or which faces share an edge. The four challenges increase in difficulty.
Does everyone get the same puzzle each day?
Yes. Cublii uses a date-based system so every player worldwide sees the same four challenges each day. This makes daily scores directly comparable.
Does Cublii collect any data or track me?
No. Cublii has no servers, no accounts, no analytics, no ads, and no tracking. All game data (scores, streaks, statistics) is stored locally on your device using Apple's SwiftData framework. The only external service is Game Center, which is managed entirely by Apple.
Does Cublii require an internet connection?
No. All puzzles are generated on-device and all gameplay happens offline. Game Center leaderboard syncing requires a connection but is not required for gameplay.
What is the difference between free and Cublii Pro?
The free version gives you the full daily puzzle (all four challenges) every day with Game Center leaderboards and shareable results. Cublii Pro ($2.99, one-time purchase) unlocks the puzzle archive (play any past day), practice mode with unlimited random puzzles, detailed statistics by question type, and visual themes. There are no subscriptions and no ads in either version.
When does the daily puzzle reset?
A new set of four challenges is available every day at midnight in your local time zone.
What are the 11 cube nets?
There are exactly 11 distinct ways to unfold a cube into a flat pattern (called hexominoes). Cublii uses all 11 of these mathematically proven shapes to generate an unlimited variety of puzzles. Each net requires a different mental folding strategy.
Is the game colorblind accessible?
Yes. Each cube face uses both a distinct color and a unique border pattern (solid, dashed, dotted, etc.). A colorblind mode is available in settings that emphasizes the border patterns for players who have difficulty distinguishing colors.
How does Game Center work with Cublii?
If you're signed into Game Center, your daily scores are automatically submitted to the daily leaderboard. You can view leaderboards and achievements through the Game Center button in the app. Game Center sign-in is optional — you can play without it.
What happens to my data if I delete the app?
All Cublii data is stored locally on your device. When you delete the app, all game data (scores, streaks, statistics) is permanently removed by iOS. Game Center achievements and leaderboard entries are retained by Apple in your Game Center profile and will reappear if you reinstall.
What iOS version is required?
Cublii requires iOS 17 or later and is designed for iPhone.
